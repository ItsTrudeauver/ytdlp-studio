# Help bubble copy standard

Use outcome-based copy.

Bad:

> Concatenation uses compatible streams.

Good:

> Downloads playlist items and tries to make one long video instead of many separate files.

Rules:

- Explain what changes in the output file.
- Mention risks only when the user can act on them.
- Avoid yt-dlp flag names unless the field is explicitly a command/advanced field.
- Keep each bubble under two short sentences.
